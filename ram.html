Datorns olika komponenter


Ram

RAM eller Random Access Memory är det minnet i datorn som lagrar data som försvinner när datorn stängs av. T.ex. så lagras operativsystemet i RAM-minnet och om du surfar så lagras datan där tills du stänger av datorn eller slutar surfa då datan raderas från RAM-minnet. Därför all data inte lagras i hårddisken eller SSD:n är för att det skulle gå för långsamt om det lagrades där, istället behövs ett RAM-minne så data mellan minnet och processorn sker snabbt.

Faktorerna som påverkar hur bra RAM-minnet presterar är RAM-minnets storlek och snabbhet. Desto större minne desto mer program kan köras samtidigt och desto snabbare RAM-minnet är desto snabbare blir datorn. Ett vanligt problem med gamla datorer är att dom blir sega, det kan bero på att för många program körs samtidigt och RAM-minnet är för litet. För att fixa det problemet kan man titta på aktivitetshanteraren och ta bort program som körs i onödan.
